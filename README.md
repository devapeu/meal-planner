# Meal Planner

Weekly meal planner with a shopping list and recipes to add from made with Vue 3 and a simple PHP + MySQL backend.

## Backend

Run SQL from `/backend/database.sql`

## Frontend

### Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev -- --host
```

### Compile and Minify for Production

```sh
npm run build
```
